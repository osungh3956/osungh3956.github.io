개인정보 보호정책

요약: 당사는 쿠키를 사용하지 않으며 어떠한 개인 데이터도 수집하지 않습니다.

웹사이트 방문자 #

  • 개인 정보를 수집하지 않습니다.
  • 브라우저에 정보를 저장하지 않습니다.
  • 제3자와 정보를 공유하거나, 전송하거나, 판매하지 않습니다.
  • 광고 회사와 정보를 공유하지 않습니다.
  • 개인 및 행동 경향을 위해 정보를 채굴하거나 수집하지 않습니다.
  • 정보를 수익화하지 않습니다.

수집하는 정보와 사용 목적 #

getdoks.org에서는 Plausible 분석을 실행합니다. 다음 정보가 수집됩니다:

  • 페이지 URL. 이 웹사이트에서 각 페이지 조회의 페이지 URL을 추적합니다. 이를 통해 어떤 페이지가 조회되었고 특정 페이지가 몇 번 조회되었는지 이해합니다. 예: https://getdoks.org/.
  • HTTP 리퍼러. 다른 사이트의 링크를 통해 이 웹사이트로 유입된 방문자 수를 파악하기 위해 리퍼러 문자열을 사용합니다. 예: https://github.com/.
  • 브라우저. 이 웹사이트를 방문할 때 사용하는 브라우저를 이해하기 위해 사용합니다. 이는 User-Agent HTTP 헤더에서 파생됩니다. 전체 User-Agent는 폐기됩니다. 예: Chrome.
  • 운영 체제. 이 웹사이트를 방문할 때 사용하는 운영 체제를 이해하기 위해 사용합니다. 운영 체제의 브랜드만 사용하고 버전 번호나 기타 세부 정보는 포함하지 않습니다. 이는 User-Agent HTTP 헤더에서 파생됩니다. 전체 User-Agent는 폐기됩니다. 예: GNU/Linux.
  • 기기 유형. 이 웹사이트를 방문할 때 사용하는 기기를 이해하기 위해 사용합니다. 이는 window.innerWidth에서 파생됩니다. 브라우저의 실제 너비(픽셀)는 폐기됩니다. 예: Desktop.
  • 방문자 국가. IP 주소를 사용하여 방문자의 국가를 확인합니다. 원산지 국가보다 더 세부적인 정보는 추적하지 않으며 방문자의 IP 주소는 폐기됩니다. IP 주소는 데이터베이스나 로그에 절대 저장하지 않습니다. 예: Canada.
